Image Snap - 1.0.0
==========

jQuery plugin that snaps images to the paragraph line height.

Also does some responsive stuff.

Eventually...


## Quick Start
Do this.

```js
$( 'p img' ).imageSnap( 'data-src' );
```

Then this.

```html
<img data-src='url.png'>
```
